# Movie Database Project

## Overview
This project is a movie database application developed as a collaborative student project. The application allows users to store, view, and manage information about movies in a structured and organized way.

The goal of this project was to practice database design, data management, and basic application development concepts.

## Features
- Add new movies to the database  
- View a list of stored movies  
- Search or filter movies by selected criteria  
- Update existing movie records  
- Delete movies from the database  

## Technologies Used
- Languages -> HTML, JavaScript, PHP, MySQL
- Database: phpMyAdmin 
- Tools -> Visual Studio Code, XAMPP

## Bonus

A user interface is included to demonstrate how this data would be used in a practical setting.
